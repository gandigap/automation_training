Задания из раздела "Additional Unit" являются вспомогательными для курса "Коллекции". В процессе изучения разделов курса рекомендуется решить 3-5 задач из списка.
Task_1.   Ввести строки из файла, записать в список. Вывести строки в файл в обратном порядке.
Task_2.   Создать список из элементов каталога и его подкаталогов.
Task_3.   Занести стихотворение в список. Провести сортировку по возрастанию длин строк.
